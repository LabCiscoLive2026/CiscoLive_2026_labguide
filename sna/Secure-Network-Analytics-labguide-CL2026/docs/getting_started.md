# Getting Started

## Access Requirements

**Connectivity and Hardware**

- **Network Access:** Please ensure you have a stable internet connection for the duration of the session.
- **Recommended Hardware:** A laptop or tablet is recommended to ensure full compatibility and optimal interaction with the dashboard interfaces.
- **VPN Client:** Cisco Secure Client
- **Terminal Emulator:** PuTTY
- **Browser:** Google Chrome

**SNA Virtual Appliances – Provided in this lab**.

- SNA Data Node (v7.6.0)
- SNA Flow Collector (v7.6.0)
- SNA Manager (v7.6.0)
